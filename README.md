markdown# Calculadora de Valor por Hora para Freelancer

Um projeto prático desenvolvido para ajudar profissionais autônomos a calcularem de forma justa e realista o valor ideal de sua hora de trabalho, com base em suas metas financeiras e rotina semanal.

## Tecnologias Utilizadas

- **HTML5**: Estruturação semântica do formulário e blocos de conteúdo.
- **CSS3**: Estilização responsiva, layout centralizado com Flexbox e efeitos visuais básicos.
- **JavaScript (ES6)**: Lógica de negócio, captura de eventos (`onclick`), validação de dados e manipulação do DOM.

## O que eu aprendi neste projeto?

Este projeto marcou o meu início na integração do JavaScript com o ecossistema Web (HTML/CSS). Durante o desenvolvimento, pratiquei:
- **Manipulação do DOM**: Capturar valores de inputs de texto/número e exibir resultados dinamicamente na tela.
- **Tratamento de dados**: Conversão de strings para números (`Number()`) para evitar erros em operações matemáticas.
- **Validação de formulários**: Criação de alertas de erro caso o usuário tente submeter campos vazios.
- **Formatação de moedas**: Uso do método nativo `toLocaleString('pt-BR')` para exibir valores no padrão monetário brasileiro (R$).
